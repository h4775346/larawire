## Livewire Starter Kit

This Project has been created to make it easy for you to start any project

- Livewire Installed.
- Custom Dialogs.
- Livewire Data tables.
- SweetAlert2 Installed.
- Livewire Modal Created.
- Roles And Permissions Included.
- Dark Mode Supported.
- Tailwind Included.
- Jetstream Used
